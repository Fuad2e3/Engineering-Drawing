## 📖 Overview
This repository contains the professional engineering drawings and architectural plans for the **Plantive** ecosystem. Developed by **Team Softece** at **Green University of Bangladesh**, these designs provide the technical blueprint for the smart gardening infrastructure, including spatial layouts for nurseries, community garden zones, and integrated hardware placement for the AI-powered care system.

---

## 📐 Key Drawings

The project consists of high-precision CAD models organized into three primary files:

- 🏛️ **[2D Plan](drawings/2D_Plan.dwg):** Detailed floor plans and spatial layouts for gardening zones.
- 🧊 **[3D Perspective](drawings/3D_Perspective.dwg):** 3D modeling and visualization of the garden infrastructure.
- 📋 **[Project Master](drawings/Project_Master.dwg):** The comprehensive technical drawing combining all structural elements.
